## 1. keyof, typeof 연산자
~~~
typeof 는 주어진 식의 값이 null인지, undefined인지, number인지 나타내는 역할을 하는 연산자입니다.
뎁스가 깊어도 동작합니다.
~~~
~~~typescript
const a = {
  "name": "John",
  "age": 30,
  "address": {
    "street": "Main Street",
    "city": "New York",
    "state": "NY"
  }
}

/**
 * (property) "address": {
    street: string;
    city: string;
    state: string;
}
 */
typeof a.address;
~~~

~~~
keyof 연산자는 객체의 타입을 받아 key로 구서오딘 문자열 또는 숫자 유니온 타입을 생성합니다.
~~~

~~~typescript
// keyof은 타입에서만 동작해서 typeof 필요 -> key를 상수타입처럼 써서 enum 대체가능
type TAtype =  typeof a;
type attr = keyof TAtype; // type attr = "name" | "age" | "address";

// enum 처럼 사용하는 방법
type keyEnum = typeof a[attr];
~~~
